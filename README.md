# TDS .NET Take Home Solution
1. After cloning the repository, open the solution in terminal and cd to *CarParkAPI*.
2. Type in *dotnet publish*.
3. In file explorer proceed to "..\tds_dotnet_assignment\CarParkMgmt\CarParkAPI\bin\Release\net8.0\publish".
4. A couple of settings to play with before starting the program can be found in appsettings json.There one can set the amount of parking spaces, the coefficient for free / occupied spaces ratio at initialization with fake data, the additional charge time span and the pricing of the additional charge.
5. In publish directory, start *CarParkAPI.exe*.
6. In browser, paste http://localhost:5000/swagger .
7. Have a nice day ...
